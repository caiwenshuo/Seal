# Seal

封印

### 开发配置
本应用是在Android Studio 3.1上面开发的。

安卓版本：
minSdkVersion 16
targetSdkVersion 23

### 功能

封印任意已安装app，设置解封时间。
解封时间app无法打开，时间到自动解锁。
